Fonctionnalités

Tout ce dont vous avez besoin,
dans un seul outil.

Conçu par un enseignant pour les enseignants. Chaque module répond à un vrai besoin du quotidien.

🎓
Conseil de classe & Bulletins
Importez vos PDF de bulletins, visualisez les moyennes T1/T2/T3, suivez l'évolution de chaque élève et rédigez vos appréciations.
👁️
Suivi de classe
Notez les observations de comportement en temps réel, gérez les absences, les travaux non rendus et les alertes par élève.
📓
Carnet de bord
Rédigez vos fiches de séance avec objectifs, activités et devoirs. Éditeur rich text complet avec polices et formatage.
🗺️
Plan de classe
Organisez visuellement votre salle, placez vos élèves par drag & drop et utilisez le mode Séance pour annoter en direct.
📅
Progression annuelle
Planifiez vos séquences par trimestre, associez des évaluations et visualisez votre avancement sur l'année.
🖊️
Tableau blanc & Étude de doc
Un tableau blanc avec widgets (minuteur, tirage au sort, QR code, feu tricolore) et un annotateur de documents PDF.
👨‍👩‍👧
Rencontres parents-profs
Planifiez et archivez vos rencontres avec les familles, avec résumé, décisions prises et alertes dashboard.
🔒
100% local, sans serveur
Vos données restent sur votre machine. Aucune inscription, aucun compte, aucune connexion obligatoire. Sauvegarde en JSON.

Téléchargement

Deux versions,
deux usages.

ClassPro existe en version Desktop pour votre machine personnelle, et en version HTML à ouvrir directement dans la classe.

💻 Version Desktop
Chez vous,
sur votre machine

L'application native à installer sur votre ordinateur personnel. Elle vous permet de gérer l'année complète et de générer tous vos documents officiels.

  • Génération de PDF bulletins, carnet, progressions
  • Import de vos bulletins Pronote (PDF)
  • Gestion complète des élèves et classes
  • Conseil de classe avec comparatif T1/T2/T3
  • Export JSON pour synchroniser avec la version HTML
  • Mac (Intel & Apple Silicon) et Windows 10/11
🍎 macOS (Apple Silicon) 🍎 macOS (Intel x64) 🪟 Windows 10/11
⚠️ macOS : si alerte développeur non identifié → Réglages Système → Confidentialité → Ouvrir quand même.
⚠️ Windows : cliquer "Informations complémentaires" puis "Exécuter quand même".
🌐 Version HTML
En classe,
sans installation

Un fichier HTML à ouvrir dans n'importe quel navigateur. Idéal pour l'usage quotidien en classe — même sans connexion internet.

  • Tableau blanc avec minuteur, tirage au sort, QR code
  • Étude et annotation de documents PDF
  • Plan de classe interactif (mode Séance en direct)
  • Suivi de classe et observations en temps réel
  • Carnet de bord et fiches de séance
  • Fonctionne hors connexion, aucune installation
Télécharger le fichier HTML Voir toutes les releases →
💡 Importez votre JSON exporté depuis la version Desktop pour retrouver toutes vos données en classe.
