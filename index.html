  • Génération de PDF bulletins, carnet, progressions
  • Import de vos bulletins Pronote (PDF)
  • Gestion complète des élèves et classes
  • Conseil de classe avec comparatif T1/T2/T3
  • Export JSON pour synchroniser avec la version HTML
  • Mac (Intel & Apple Silicon) et Windows 10/11

Installation

Installer ClassPro
en 2 minutes

Votre système d'exploitation va afficher une alerte de sécurité — c'est normal. Voici exactement quoi faire.

1
Téléchargez le fichier .dmg
Cliquez sur le bouton macOS dans la section téléchargement ci-dessus. Le fichier ClassPro-Desktop-x.x.x-arm64.dmg (Apple Silicon) ou x64.dmg (Intel) se télécharge dans votre dossier Téléchargements.
2
Ouvrez le fichier .dmg
Double-cliquez sur le fichier téléchargé. Une fenêtre s'ouvre, glissez l'icône ClassPro dans le dossier Applications.
3
macOS bloque l'ouverture — c'est normal
Quand vous double-cliquez sur ClassPro pour la première fois, macOS affiche cette alerte :
🚫
"ClassPro" ne peut pas être ouvert car Apple ne peut pas vérifier qu'il ne contient pas de logiciel malveillant.
Ce logiciel doit être mis à jour. Contactez le développeur pour obtenir une version compatible avec cette version de macOS. Si vous êtes sûr de sa provenance, vous pouvez tout de même l'ouvrir.
Annuler
OK
Cliquez OK pour fermer cette alerte — ne cliquez pas "Déplacer vers la Corbeille".
4
Autorisez ClassPro dans les Réglages
Ouvrez Réglages Système (l'icône engrenage) → Confidentialité et sécurité → faites défiler jusqu'à la section Sécurité. Vous verrez le message "ClassPro a été bloqué" avec un bouton "Ouvrir quand même" → cliquez dessus → confirmez avec votre mot de passe.
5
ClassPro s'ouvre normalement
Relancez ClassPro depuis Applications. Cette manipulation n'est à faire qu'une seule fois — les prochaines ouvertures seront directes.
💡
Cette alerte apparaît parce que ClassPro n'est pas signé avec un certificat Apple payant. Le logiciel est sûr — il est développé par un enseignant et son code source est disponible sur GitHub.
1
Téléchargez le fichier .exe
Cliquez sur le bouton Windows 10/11 dans la section téléchargement. Le fichier ClassPro-Desktop-Setup-x.x.x.exe se télécharge dans votre dossier Téléchargements.
2
Windows Defender bloque le fichier — c'est normal
Quand vous double-cliquez sur le fichier, Windows affiche cet écran bleu :
Windows Defender SmartScreen
🛡️
Windows a protégé votre ordinateur
Windows SmartScreen a empêché le démarrage d'une application non reconnue. L'exécution de cette application peut mettre en danger votre PC.

Application : ClassPro-Desktop-Setup.exe
Éditeur : Éditeur inconnu
Ne pas exécuter
Informations complémentaires
3
Cliquez "Informations complémentaires"
Un lien bleu "Informations complémentaires" apparaît en bas de l'écran bleu. Cliquez dessus — la fenêtre se met à jour et affiche un nouveau bouton.
4
Cliquez "Exécuter quand même"
Un bouton "Exécuter quand même" apparaît en bas à droite. Cliquez dessus pour lancer l'installation.
Windows Defender SmartScreen
🛡️
Windows a protégé votre ordinateur
Application : ClassPro-Desktop-Setup.exe
Éditeur : Éditeur inconnu
Ne pas exécuter
Exécuter quand même
5
Suivez l'assistant d'installation
L'installeur se lance normalement. Cliquez Suivant puis Installer. ClassPro apparaît dans votre menu Démarrer et sur le bureau.
💡
Windows Defender affiche cette alerte pour tout logiciel qui n'a pas de certificat de signature numérique payant. ClassPro est sûr — son code source est disponible publiquement sur GitHub.
